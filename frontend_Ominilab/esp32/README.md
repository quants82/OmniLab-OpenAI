# Public ESP32 Firmware

These MicroPython sources are intentionally open and unencrypted.

| Experiment | Main source |
| --- | --- |
| Harmonic motion with BMI160 | `esp32_shm_bmi160.py` |
| Specific heat capacity of water | `esp32_specific_heat.py` |
| Electromagnetic induction | `esp32_induction.py` |
| Capacitor charge and discharge | `esp32_capacitor.py` |
| Incandescent lamp I-V characteristic | `esp32_lamp_va.py` |
| Resistor I-V characteristic | `esp32_resistor_va.py` |

`vatli_auth.py` is retained only as a compatibility name for Wi-Fi setup and MAC identification. It contains no subscription check, ownership validation, or device secret.
